# JOB_ID

## Description

Sets a custom task ID in  TensorFlow  training and online inference scenarios.

It is recommended that the string contain only uppercase letters, lowercase letters, digits, hyphens \(-\), and underscores \(_\). Avoid using an all-numeric  **JOB_ID**  that starts with 0.

## Example

```bash
export JOB_ID=10087
```

## Constraints

None

## Applicability

Ascend 950PR/Ascend 950DT

Atlas A3 training product/Atlas A3 inference product

Atlas A2 training product/Atlas A2 inference product

Atlas training product

Atlas inference product
