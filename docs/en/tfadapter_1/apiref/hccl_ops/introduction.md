# Description

The TF Adapter provides collective communication operator APIs, which can be used together with the Python-based communicator management APIs provided by HCCL. For details about the Python-based communication domain management APIs provided by HCCL, see [Huawei Collective Communication Library \(HCCL\)](https://www.hiascend.com/document/detail/en/canncommercial/900/API/hcclug/hcclug_000001.html).
