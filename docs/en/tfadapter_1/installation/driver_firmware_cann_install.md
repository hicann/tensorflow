# Installing the Driver, Firmware, and CANN Package

Before model development, prepare a hardware environment with the  AI processor. Install the Ascend NPU driver and CANN software \(including the Toolkit and ops package\) by referring to  [CANN Installation](https://www.hiascend.com/en/cann/download?versionId=731&ids=d806%2Ch0502%2Ch0601%2Ch0702), and configure environment variables.
